# Card Info Dark Mode Text Fix (Issue #14078)\n\n## What does this do?\nAdds a dark mode `color` override to `.ease-card-info` using a lighter primary color (`#a09af8`) that maintains sufficient contrast on dark backgrounds.\n\n## How is it used?\n```html\n<div class="ease-card ease-card-info">Info content</div>\n```\n\n## Why is it useful?\nThe hardcoded `#4b44cc` (dark purple) used in light mode fails WCAG contrast requirements on dark backgrounds. The fix uses a lighter tint of the same hue that passes contrast checks in dark mode.
